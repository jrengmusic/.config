# WezTerm Configuration

This repository contains my WezTerm terminal configuration.

## Files

- `wezterm.lua` - Main WezTerm configuration file

## WezTerm Configuration

The `wezterm.lua` file includes:

- Custom color scheme with blue/cyan theme
- Kitty graphics protocol enabled for inline images
- 10,000 line scrollback buffer
- Custom key bindings
- Skip close confirmation for common shells
